◆ commence()

int commence ( const char *  entree,
  ... 
)

Teste si la chaîne donnée commence par les tokens donnés (sous forme d'une liste de chaînes de caractère terminée par un NULL).

La fonction ignore naturellement les espaces (caractères validant est_espace) et retourne l'indice après lequel le dernier token a été trouvé, ou -1 si la ligne ne commence pas par tous les tokens donnés.

Si un token se termine par \> on considère qu'il est borné, c'est à dire que le token s'arrête lorsqu'on rencontre un délimiteur. Par exemple : commencer("abc:", "abc", NULL) renvoie -1, mais commencer("abc:", "abc\\>", NULL) renvoie 3

Paramètres
entreechaîne à vérifier
Renvoie
indice de la fin du dernier token ou -1 si la chaîne ne commence pas par les token donnés
